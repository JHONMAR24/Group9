Source of the data

This data set was collected from GitHub portal. Each row of this data set represents a shot Kobe Bryant took during the five games of the 2009 NBA finals. Kobe Bryant's performance earned him the title of Most Valuable Player and many spectators commented on how he appeared to show a hot hand.

Link to download CSVfile: Here!

Definition of each variable

Variable Definition
vs A categorical vector, ORL if the Los Angeles Lakers played against Orlando
game A numerical vector, game in the 2009 NBA finals
quarter A categorical vector, quarter in the game, OT stands for overtime
time A character vector, time at which Kobe took a shot
description A character vector, description of the shot
basket A categorical vector, H if the shot was a hit, M if the shot was a miss
